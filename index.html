Explore the Power of Map and Set

What is a Map?

A Map is a collection of key-value pairs where keys can be of any data type.

const map = new Map();
map.set('name', 'Alice');
console.log(map.get('name')); // Alice

What is a Set?

A Set is a collection of unique values. It ensures no duplicate items.

const set = new Set();
set.add(1);
set.add(2);
set.add(1); // No duplicate
console.log(set); // Set {1, 2}
